TerraPrint was developed using Claude Code to create topographic models for 3D printing with a 4 color system. Colors are derived from elevation or satellite data, which is user tuneable. 

While this was developed for a personal project, it is possible to select custom areas anywhere in the world elevation data is present.

Model Settings Options:
-Vertical Exaggeration
-Base Thickness
-Resolution
-Detect Water (kinda sucks)
-Contour Steps (Terraced Layers)

Terrain Colors Settings:
setup for 4 colors - elevation thresholds can be set to user preference.

Satellite Color Settings:
4/8 color support
Detail
Contrast
Brightness
Region Shape
Texture Smoothing
Elevation cap

Edit Options:
Water painting features
Color correction features

Export Options:
-STL (Single Color - single object)
-3MF (Multicolor - multi object)

Additional Tools:
Frames/Cards/Shadow box options
