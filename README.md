TerraPrint was developed using Claude Code to tile the entire state of Oregon for 3D printing with a 4 color system. Colors are derived from elevation data, which is user tuneable. 

While this was developed for my personal Oregon project, it is possible to select custom areas anywhere in the world elevation data is present.

Model Settings Options:
-Vertical Exaggeration
-Base Thickness
-Resolution
-Detect Water (kinda sucks)
-Contour Steps (Terraced Layers)

Terrain Colors Settings:
setup for 4 colors - elevation thresholds can be set to user preference.

Export Options:
-STL (Single Color - single object)
-3MF (Multicolor - multi object)

Additional Tools:
There are tools in here for "mark printed", columns, rows, etc that pertain to my personal project. If you'd like to print Oregon or even expand the project to support custom area tiling, feel free. I may implement it myself at some point.

The Import and Export buttons at the top are used for saving a hard copy of my progress and selected settings.
