TerraPrint
Turn real terrain into multi-color 3D-printable models
  1. Find a place — search by name or pan the map, then draw a rectangle or circle with the tools on the right.
  2. Pick a color source (Imagery & Color) — elevation bands for classic hypsometric tints, or satellite / land-cover for photographic color. 3MF exports print each color as its own filament (AMS/MMU).
  3. Generate — elevation is fetched from USGS/Terrarium data at up to ~10 m detail and meshed into a watertight model. Check the size readout in Selected Area.
  4. Refine — Paint Water fixes missed lakes, Color Correct repaints misclassified regions, and Composition adds frames, engraved edges, or a display card.
  5. Export — 3MF for multi-color printing (Bambu/Orca/Prusa slicers), STL for single color. Big area? Use Tiling to split it into full-resolution pieces.

Settings persist between visits. Reopen this guide anytime with the ? button in the header.
